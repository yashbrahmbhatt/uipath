NAME:
   uipath du discovery tags - Retrieve tags

USAGE:
   uipath du discovery tags [command options] [arguments...]

DESCRIPTION:
   Retrieve all tags from a project. 

   Required scopes: <b>Du.Digitization.Api or Du.Classification.Api or Du.Extraction.Api or Du.Validation.Api</b></br>

OPTIONS:
   --api-version string (default: 1.0)
      Version of the APIs.

      Allowed values:
      - 1.0

   --project-id string (default: 00000000-0000-0000-0000-000000000000)
      ID of the Project.
