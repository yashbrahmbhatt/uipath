NAME:
   uipath orchestrator buckets get-by-id - Gets a single Bucket.

USAGE:
   uipath orchestrator buckets get-by-id [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: OR.Administration or OR.Administration.Read.

   Required permissions: Buckets.View.

OPTIONS:
   --folder-id integer (required)
      Folder/OrganizationUnit Id

   --key integer (required)

   --expand string
      Indicates the related entities to be represented inline. The maximum depth is 2.

   --select string
      Limits the properties returned in the result.
