NAME:
   uipath orchestrator execution-media download-media-by-job-id - Downloads execution media by job id

USAGE:
   uipath orchestrator execution-media download-media-by-job-id [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: OR.Monitoring or OR.Monitoring.Read.

   Required permissions: ExecutionMedia.View.

OPTIONS:
   --folder-id integer (required)
      Folder/OrganizationUnit Id

   --job-id integer (required)
