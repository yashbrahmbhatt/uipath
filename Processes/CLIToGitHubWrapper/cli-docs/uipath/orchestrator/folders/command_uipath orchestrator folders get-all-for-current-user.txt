NAME:
   uipath orchestrator folders get-all-for-current-user - Returns a subset (paginated) of the folders the current user has access to.
                                                          The response will be a list of folders;

USAGE:
   uipath orchestrator folders get-all-for-current-user [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: OR.Folders or OR.Folders.Read.

   Requires authentication.

OPTIONS:
   --skip integer

   --take integer
