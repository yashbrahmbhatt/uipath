NAME:
   uipath orchestrator machines post - Creates a new machine.

USAGE:
   uipath orchestrator machines post [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: OR.Machines or OR.Machines.Write.

   Required permissions: Machines.Create.

OPTIONS:
   --name string (required)
      The name of the Machine a Robot is hosted on.

   --automation-cloud-slots integer
      Number of AutomationCloud slots to be reserved at runtime

   --automation-cloud-test-automation-slots integer
      Number of AutomationCloudTestAutomation slots to be reserved at runtime

   --automation-type string
      The automation type the machine supports.

      Allowed values:
      - Any
      - Foreground
      - Background

   --client-secret string
      Client secret generated by Identity Server
      Use in combination with LicenseKey (ClientId) to connect robot

   --description string

   --endpoint-detection-status string
      The EDR protection status of the connected hosts.

      Allowed values:
      - NotAvailable
      - Mixed
      - Enabled

   --headless-slots integer
      Number of Headless slots to be reserved at runtime

   --hosting-slots integer
      Number of Hosting slots to be reserved at runtime

   --id integer

   --key string
      An immutable unique identifier that is preserved during tenant migration

   --license-key string
      The key is automatically generated from the server for the Robot machine.
      <para />For the robot to work, the same key must exist on both the robot and Orchestrator.
      <para />All robots on a machine must have the same license key in order to register correctly.

   --maintenance-window object
      Represents a maintenance window for a cloud machine

      Example:
         cronExpression=string; duration=integer; enabled=boolean; jobStopStrategy=string; nextExecutionTime=string; timezoneId=string

   --non-production-slots integer
      Number of NonProduction slots to be reserved at runtime

   --robot-users object (multiple)
      Robots assigned to template machine.

      Example:
         HasTriggers=boolean; RobotId=integer; UserName=string

   --robot-versions object (multiple)
      The versions of the Robots hosted on the Machine.

      Example:
         Count=integer; MachineId=integer; Version=string

   --scope string
      The scope of the Machine (Default / Shared / PW / Cloud / Serverless).

      Allowed values:
      - Default
      - Shared
      - PersonalWorkspace
      - Cloud
      - Serverless
      - AutomationCloudRobot
      - ElasticRobot
      - ProcessOrchestration
      - AgentService

   --serverless-licensing-model string
      The type of serverless licensing model

      Allowed values:
      - RobotUnits
      - LicenseSlots

   --tags object (multiple)

      Example:
         DisplayName=string; DisplayValue=string; Name=string; Value=string

   --target-framework string
      The target frameworks the machine supports.

      Allowed values:
      - Any
      - Windows
      - Portable

   --test-automation-slots integer
      Number of TestAutomation slots to be reserved at runtime

   --type string
      The type of the Machine (Standard / Template).

      Allowed values:
      - Standard
      - Template

   --unattended-slots integer
      Number of Unattended slots to be reserved at runtime

   --update-policy object
      The policy applied for update robots' versions

      Example:
         SpecificVersion=string; Type=string

   --vpn-settings object

      Example:
         cidr=string
