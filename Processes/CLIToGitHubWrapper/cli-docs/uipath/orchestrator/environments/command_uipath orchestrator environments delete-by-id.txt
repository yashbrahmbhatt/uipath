NAME:
   uipath orchestrator environments delete-by-id - Deletes an environment.

USAGE:
   uipath orchestrator environments delete-by-id [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: OR.Robots or OR.Robots.Write.

   Required permissions: Environments.Delete.

OPTIONS:
   --folder-id integer (required)
      Folder/OrganizationUnit Id

   --key integer (required)
