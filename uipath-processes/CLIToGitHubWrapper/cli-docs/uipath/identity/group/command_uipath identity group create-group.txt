NAME:
   uipath identity group create-group - Creates a new local group

USAGE:
   uipath identity group create-group [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: PM.Group or PM.Group.Write.

OPTIONS:
   --directory-user-member-ids string,string,...
      IDs of the users to be added as members of the group.

   --id string
      Group unique identifier

   --name string
      Group Name

   --partition-global-id string
      Group partition identifier
