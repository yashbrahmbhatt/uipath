NAME:
   uipath identity user change-password - Changes the user's password.

USAGE:
   uipath identity user change-password [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: PM.User or PM.User.Write.

OPTIONS:
   --user-id string (required)
      The database user id.

   --new-password string
      The user's new password.

   --old-password string
      The user's old password.
