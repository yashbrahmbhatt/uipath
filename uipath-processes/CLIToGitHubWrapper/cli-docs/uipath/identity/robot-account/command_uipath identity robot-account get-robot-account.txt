NAME:
   uipath identity robot-account get-robot-account - Get robot account by partition and robot account Id

USAGE:
   uipath identity robot-account get-robot-account [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: PM.RobotAccount or PM.RobotAccount.Read.

OPTIONS:
   --partition-global-id string (required)

   --robot-account-id string (required)
