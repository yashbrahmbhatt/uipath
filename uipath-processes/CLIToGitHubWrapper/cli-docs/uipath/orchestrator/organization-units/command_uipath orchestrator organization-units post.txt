NAME:
   uipath orchestrator organization-units post - Creates an organization unit.

USAGE:
   uipath orchestrator organization-units post [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: OR.Folders or OR.Folders.Write.

   Required permissions: Units.Create.
   DEPRECATED: 
   Kept for backwards compatibility. Use Post from FoldersController  instead
   Please refer to https://docs.uipath.com/orchestrator/reference

OPTIONS:
   --display-name string (required)
      The name of the organization unit.

   --id integer
