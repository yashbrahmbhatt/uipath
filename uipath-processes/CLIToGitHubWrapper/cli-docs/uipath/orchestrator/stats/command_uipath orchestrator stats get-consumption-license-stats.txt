NAME:
   uipath orchestrator stats get-consumption-license-stats - Gets the consumption licensing usage statistics

USAGE:
   uipath orchestrator stats get-consumption-license-stats [command options] [arguments...]

DESCRIPTION:
   OAuth required scopes: OR.Monitoring or OR.Monitoring.Read.

   Required permissions: License.View.

OPTIONS:
   --days integer
      Number of reported license usage days

   --tenant-id integer
      The Tenant's Id - can be used when authenticated as Host
